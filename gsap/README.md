# GSAP 이란?
`The GreenSock Animation Platform`의 줄임말으로 타임라인 기반의 애니메이션 자바스크립트 라이브러리이다.

## Tween 이란
Tween 은 모든 애니메이션 작업을 수행한다. 애니메이션을 적용하려는 객체, 기간 및 애니메이션을 적용하려는 속성을 입력한 다음, Tween 의 재생 헤드가 새 위치로 이동하면, 해당 지점에서 속성값이 무엇인지 파악하고 그에 따라 적용한다.   

종류
- gsap.to() : to() 메서드는 타겟 엘리먼트의 끝 값을 지정하는 가장 기본적인 메서드로 애니메이션 시작 후 종료되는 값을 지정해주는 메서드이다.
- gsap.from() : from() 메서드는 to() 메서드와는 반대로 시작점을 지정할 수 있다.
- gsap.fromTo() : formTo()는 타겟 엘리먼트의 시작과 끝을 함께 지정해주는 메서드이다.

`delay`를 통해 기본 시퀀싱을 사용할 수 있지만, 타임라인을 사용하면 시퀀싱 및 복잡한 안무를 훨씬 쉽게 수행할 수 있다.



## Timeline 이란
Timeline 은 Tweens 의 컨테이너이다. 애니메이션을 원하는 시간에 배치한 다음 pause(), play(), progress(), reverse(), timeScale() 등과 같은 메서드를 사용하여 전체 시퀀스를 쉽게 제어할 수 있는 시퀀싱 도구이다.    
모든 애니메이션(Tween, Timeline)은 상위 타임라인(globalTimeline)에 배치된다. 

``` text
                        PLAYHEAD
|--------------timeline-----|-----------|
|--tween1--|                |
           |-----tween2-----|-----------|

```
