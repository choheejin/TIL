# GSAP 이란?
`The GreenSock Animation Platform`의 줄임말으로 타임라인 기반의 애니메이션 자바스크립트 라이브러리이다.

## Tween 이란
Tween 은 모든 애니메이션 작업을 수행한다. 애니메이션을 적용하려는 객체, 기간 및 애니메이션을 적용하려는 속성을 입력한 다음, Tween 의 재생 헤드가 새 위치로 이동하면, 해당 지점에서 속성값이 무엇인지 파악하고 그에 따라 적용한다.   

종류
- gsap.to()
- gsap.from()
- gsap.fromTo()

`delay`를 통해 기본 시퀀싱을 사용할 수 있지만, 타임라인을 사용하면 시퀀싱 및 복잡한 안무를 훨씬 쉽게 수행할 수 있다.
